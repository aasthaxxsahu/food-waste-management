- ## Project Overview

The Local Food Wastage Management System is designed to reduce food wastage by connecting food providers such as restaurants, grocery stores, and individuals with receivers including NGOs and people in need.

The system enables food listing management, claim tracking, contact discovery, data analysis, and CRUD operations through an interactive Streamlit dashboard connected to a Railway MySQL database.

## Project Structure

- app.py → Main Streamlit application
- db_connection.py → Database connection module
- Food_Waste_Queries.sql → SQL query collection
- CRUD_Operations.sql → CRUD operation scripts
- *.csv → Dataset files
- food_waste_eda.ipynb → EDA and visualization notebook
